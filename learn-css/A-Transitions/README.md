# Lesson 1: What are Transitions?

**Section A: Transitions** | Lesson 1 of 6

## 🎯 Learning Objectives

By completing this lesson, you will be able to:

- Understand how CSS transitions work
- Write transition declarations

## 📚 Key Concepts

- Using CSS pseudo-classes (`:hover`, `:active`, `:focus`)
- The `transition` property and its syntax

## 📖 Lesson Overview

Transitions allow CSS properties to change smoothly over a specified duration instead of changing instantly. They enhance user experience by providing visual feedback and making interfaces feel more polished and responsive.
