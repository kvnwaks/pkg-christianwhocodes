## 📚 Key Concepts

- Using `from` and `to` keywords

## 📖 Lesson Overview

CSS animations use `@keyframes` to define multi-step sequences that can run automatically without user interaction. Unlike transitions which animate between two states, animations can have multiple intermediate steps and loop indefinitely.
