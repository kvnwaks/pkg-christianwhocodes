# Learn CSS

A comprehensive, hands-on course to master CSS animations, transitions, and transforms through progressive learning units.

## 🎯 What You'll Learn

By the end of this course, you will:

- Master CSS transitions, animations, and transforms
- Apply these skills to real-world web design scenarios
- Create smooth, performant animations compatible across browsers
- Build muscle memory through hands-on challenges and exercises

## 📚 Course Structure

This course is organized into **4 progressive sections** with **22 learning units**. Each unit includes:

- **Lesson Transcript**: Step-by-step explanation of concepts
- **Code Examples**: `begin.html/css` → `end.html/css` showing the evolution
- **Challenges**: Practice exercises to reinforce learning

### How to Use This Course

1. Start with Section A and progress sequentially
2. Read the lesson transcript in each folder
3. Study the `begin` files to understand the starting point
4. Follow along or try implementing yourself
5. Compare with `end` files to see the solution
6. Complete challenges to test your understanding

## 🚀 Getting Started

**New to this course?** Start here:

- 📘 [Complete Getting Started Guide](./GETTING_STARTED.md) - Setup, tips, and how to use this course
- 📑 [Table of Contents](./TABLE_OF_CONTENTS.md) - Full course outline with quick navigation
- 📊 [Progress Tracker](./PROGRESS_TRACKER.md) - Track your learning journey

**Quick Start:**

1. **Clone this repository**

   ```bash
   git clone https://github.com/christianwhocodes/learn-css-animations.git
   cd learn-css-animations
   ```

2. **Start with Section A, Lesson 1**

   ```bash
   cd A-Transitions/01-What-are-Transitions
   ```

3. **Open the files in your browser and code editor**
   - Read the markdown transcript
   - Open `begin.html` in your browser
   - Open `begin.css` in your code editor
   - Follow along with the lesson

## 💡 Learning Tips

- **Build Muscle Memory**: Early challenges may seem simple, but they're designed to help you internalize the patterns
- **Real-World Focus**: Each section ends with a practical challenge applying skills to realistic scenarios
- **Incremental Progress**: Each lesson builds on previous ones, so follow the sequence
- **Experiment**: Try modifying the examples to see how properties interact
